# QuizMaster

An interactive quiz web application built with React. Users can attempt multiple-choice questions and get instant feedback with real-time scoring.

## Installation

Clone the repository and install dependencies:

```bash
git clone <your-repository-url>
cd <repository-name>
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```


## Features

- Multiple-choice quizzes
- Real-time score tracking
- Instant feedback on answers
- Responsive design for mobile and desktop

## Technologies

- React
- Vite
- JavaScript
- CSS
